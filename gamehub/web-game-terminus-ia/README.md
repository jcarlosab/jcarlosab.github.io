# TERMINUS // SPACE SECTOR

Shooter espacial con estetica de terminal retro implementado con HTML, CSS y JavaScript vanilla. Rendering por Canvas.

## Como jugar

1. Controla la nave espacial esquivando asteroides y enemigos
2. Sobrevive el mayor numero de oleadas posible
3. Recoge puntos por cada oleada superada
4. El juego termina si la nave es impactada

## Controles

- **Flechas / WASD**: Mover la nave
- **Touch & Drag**: Control tactil en movil (joystick virtual)

## Caracteristicas

- Sistema de oleadas con dificultad progresiva
- Dos tipos de enemigos (diver y zigzag)
- Efectos de particulas (explosiones, estelas, escombros)
- Shake de pantalla en impactos
- Deteccion de near-miss con texto flotante
- Estetica visual de terminal retro (verde sobre negro)

## Archivos

- `index.html` - Estructura del juego
- `game.js` - Logica del juego
- `style.css` - Estilos visuales
