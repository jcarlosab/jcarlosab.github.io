# Torres de Hanoi

Juego clasico de las Torres de Hanoi implementado con HTML, CSS y JavaScript vanilla.

## Como jugar

1. Haz clic en una torre para recoger la ficha superior
2. Haz clic en otra torre para colocarla
3. Solo puedes colocar fichas mas pequenas sobre fichas mas grandes
4. Mueve todas las fichas a la torre 3

## Controles

- **Click izquierdo**: Recoger o soltar ficha

## Archivos

- `index.html` - Estructura del juego
- `index.js` - Logica del juego
- `style.css` - Estilos visuales
