# Memory

Juego de emparejar cartas con imagenes pixel-art implementado con HTML, CSS y JavaScript vanilla.

## Como jugar

1. Haz clic en una carta para voltearla
2. Voltea otra carta para intentar encontrar la pareja
3. Si coinciden, se quedan boca arriba; si no, se voltean de nuevo
4. Encuentra todas las parejas para terminar la partida

## Puntuacion

- Cada pareja suma 1000 puntos (maximo 9000)
- Despues del movimiento 15, cada movimiento extra penaliza -250 puntos

## Controles

- **Click izquierdo**: Voltear carta
- **Mute**: Silenciar sonidos
- **Reglas**: Ver las reglas del juego
- **Reiniciar**: Empezar una nueva partida

## Archivos

- `index.html` - Estructura del juego
- `js/main.js` - Logica del juego
- `css/style.css` - Estilos visuales
- `assets/` - Imagenes y sonidos
