# Slide Puzzle

Rompecabezas deslizante implementado con HTML, CSS y JavaScript vanilla. Multiples imagenes y niveles de dificultad.

## Como jugar

1. Selecciona un puzzle y nivel de dificultad (3x3 Facil o 4x4 Medio)
2. Haz clic en una ficha adyacente al espacio vacio para deslizarla
3. Ordena las fichas para reconstruir la imagen original
4. Tienes 120 segundos para completar el puzzle

## Niveles

- **Facil (3x3)**: 3 imagenes disponibles
- **Medio (4x4)**: 1 imagen disponible

## Controles

- **Click izquierdo**: Deslizar ficha
- **Reset**: Reiniciar puzzle actual

## Archivos

- `index.html` - Estructura del juego
- `js/main.js` - Logica del juego
- `css/styles.css` - Estilos visuales
- `assets/` - Imagenes y sonidos
