# GAMEHUB

Coleccion de juegos web retro implementados con HTML, CSS y JavaScript vanilla. Sin dependencias ni build.

Abrir `index.html` en el navegador para acceder a todos los juegos.

## Juegos

| Juego | Descripcion | Tecnologia |
|-------|-------------|------------|
| [Memory](web-game-memory/) | Emparejar cartas con imagenes pixel-art | HTML/CSS/JS |
| [Slide Puzzle](web-game-slidepuzzle/) | Rompecabezas deslizante (3x3 y 4x4) | HTML/CSS/JS |
| [TERMINUS](web-game-terminus-ia/) | Shooter espacial con estetica de terminal | HTML/CSS/JS + Canvas |
| [Pong](web-game-pong/) | Clasico Pong contra la IA | HTML/CSS/JS + Canvas |
| [Torres de Hanoi](web-game-hanoi/) | Puzzle interactivo de torres | HTML/CSS/JS |

## Como ejecutar

```bash
# Opcion 1: Abrir directamente en navegador
open index.html

# Opcion 2: Servidor local (opcional)
python -m http.server 8000
# Abrir http://localhost:8000
```

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Sin dependencias externas
- Sin build necesario
