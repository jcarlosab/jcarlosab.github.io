# Pong

Juego clasico Pong implementado con HTML, CSS y JavaScript vanilla. Incluye IA oponente.

## Como jugar

1. El jugador controla la paleta izquierda (blanca)
2. La paleta derecha es controlada por la IA
3. El primer jugador en llegar a 10 puntos gana

## Controles

- **W**: Mover paleta arriba
- **S**: Mover paleta abajo

## Archivos

- `index.html` - Estructura del juego
- `index.js` - Logica del juego
- `style.css` - Estilos visuales
