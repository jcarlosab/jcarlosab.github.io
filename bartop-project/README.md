# Bartop Project

Construcción de una máquina recreativa bartop utilizando Raspberry Pi 4, materiales reciclados y diversos componentes electrónicos. Diseñada con LibreCalc, SketchUp y Blender. Sistema operativo: Batocera v40.

### Software

- **LibreCalc** → Planos técnicos
- **SketchUp / Blender** → Modelado 3D
- **SO**: Batocera v40
