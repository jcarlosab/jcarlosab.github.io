# empty-wallet

App de gastos personales. Registro, visualizacion y analisis de gastos con graficas interactivas.

## Tecnologias

- **React 18** + **Vite**
- **React Router** - Navegacion SPA
- **Chart.js** + **react-chartjs-2** - Graficas interactivas
- **@libsql/client** - Base de datos
- **ESLint** + **Prettier** - Linting y formato

## Instalacion

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Build de produccion
- `npm run lint` - Verificar codigo
- `npm run format` - Formatear con Prettier
